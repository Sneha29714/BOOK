Overview

The Library Management & Recommendation System is a web-based platform designed to help students and teachers search, locate, and manage library books efficiently. The system includes an interactive library map, role-based dashboards, syllabus-wise book recommendations, and real-time visibility of recommended resources.

This project focuses on improving the academic experience by making book discovery faster, smarter, and more organized.
